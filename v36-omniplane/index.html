v36 // OMNIPLANE

The unified plane where all dimensions converge.

Omniplane คือระนาบรวมทุกมิติของจักรวาล ORIGIN เป็นพื้นที่ที่กาลเวลา มิติ และความจริงทั้งหมดสามารถเชื่อมถึงกันได้โดยตรง

หลังจาก Bioframe ก่อรูป “ชีวิตเชิงจักรวาล” ใน v35 ชั้น Omniplane ทำหน้าที่เป็นสนามที่สิ่งมีชีวิตระดับจักรวาลสามารถเคลื่อนผ่าน มิติใดก็ได้อย่างเป็นธรรมชาติ

นี่คือโครงสร้างที่ทำให้การเดินทางข้ามมิติ การซ้อนทับของเวลา และการเชื่อมโยงความจริงหลายชุดเกิดขึ้นได้จริง และจะนำไปสู่การเปิดประตูเวลาใน v37 Chronos Gate
